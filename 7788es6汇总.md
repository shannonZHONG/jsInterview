默认参数值<br>

```
function sum(a,b){
console.log('b:'+b)
}
return a+b;
sum(1)
```
返回的结果是 b:undefined <br>
传参时候：只给了a的值，而没有b的值, 同时又没有默认参数.<br>


```
  function sum(a,b){
  a = a || 0;
  b = b || 0;
  return a +b;
  }   
  sum(1);
```
返回结果是：1<br>
即使没有完整的传chan
